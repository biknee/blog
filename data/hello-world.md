## Welcome to My Blog

This is a **local blog** built with:

- **Go 1.26** + **Gin** framework
- **Markdown** file storage (no database!)
- **Neo-Brutalism** design style

### Code Example

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, Blog!")
}
```

### Features

1. Write posts in **Markdown**
2. Auto-generated slugs from titles
3. Tag-based filtering
4. Bold, brutalist UI

> Neo-Brutalism is about honest design — thick borders, hard shadows, no rounded corners.

Enjoy blogging!
